## [1.0.0] - 2026-01-07

### Added
- Initial release of LZA foundation module
- Deploy LZA installer CloudFormation stack
- Create IAM roles for Platform integration
- Store configuration in SSM Parameter Store
- Optional API Gateway for external integration
- Configurable organizational unit structure
- Support for Control Tower enabled/disabled modes
- Single account mode for testing/development
